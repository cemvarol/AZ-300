Create a user account using Azure PowerShell

 

First command will collect Tenant Details,

 

Second command will create a password profile for the user, and insert value.

Using that Password profile, you will be creating your first user in azure active directory

# Getting Tenant Details

$tenant = Get-AzureADTenantDetail

$tenantDomain = ($tenant.VerifiedDomains | Where-Object { $_._Default -eq $True }).Name

# Creating the user

$PasswordProfile = New-Object -TypeName Microsoft.Open.AzureAD.Model.PasswordProfile

$PasswordProfile.Password = "1q2w3e4r5t6y*"

New-AzureADUser -AccountEnabled $True -DisplayName "Clark Kent" -PasswordProfile $PasswordProfile -MailNickName "CK" -UserPrincipalName "superman@$tenantDomain"

Create a user account using Azure Azure CLI

First command will collect Tenant Details,

Second command will create a password profile for the user, and insert value.

Using that Password profile, you will be creating your first user in azure active directory

# Getting Tenant Details

A=$(az ad signed-in-user show --query userPrincipalName)

B=${A:$(echo `expr index "$A" @`)}

C=${B:: -1}

# Creating the user

az ad user create --display-name Batman --user-principal-name Batman@$C --password 1q2w3e4r5t6y*
